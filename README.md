# Classroom-Control
A simple classroom control app for teachers. <br>
Originally made for robotics summer camp management so you will have to modify it to work for you.

## How to use
I'm too lazy to explain so you can go and figure it out.<br><br>
In simple,
1. Make sure ALL computer (students and teacher) are connected on the same network.
2. Clients connect to teacher computers local IP address (will have to modify code).

The code automatically changes the student's wifi to match the teacher's so you will need to modify that too.
